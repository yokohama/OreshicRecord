# OreshicRecord プロジェクト概要

## 概要

OreshicRecordは、コマンドの実行記録や作業メモを保存・検索するためのプロジェクトです。
主な機能は以下の通りです:

- コマンドの実行記録の保存と検索
- 作業メモの保存と検索
- 現在の作業トラックの記録と切り替え

## 技術スタック

- Python: プロジェクトの実装言語
- TOML: 設定ファイルの記述形式

## ディレクトリ構成

- `ors`: プロジェクトのソースコード
  - `cli`: CLIパーサー
    - `parser.py`: コマンドライン引数のパーサー
  - `core`: コア機能
    - `settings.py`: 設定関連の関数群
  - `features`: 各機能の実装
    - `command.py`: コマンド実行記録関連の機能
    - `recording.py`: 記録機能の実装
    - `track.py`: トラック機能の実装
    - `writeup.py`: ライトアップ機能の実装
  - `exec`: コマンド実行関連 
    - `stream.py`: コマンド実行とストリーム処理  
  - `__main__.py`: プログラムのエントリーポイント
- `records`: 記録されたデータ
  - `commands`: コマンドの実行記録
  - `writeups`: 作業メモ
  - `tracks`: 作業トラックの記録
- `config.toml`: プロジェクトの設定ファイル

## 実装指示

- PEP8の警告はできだけ修正をする。
- 同じ操作を3回以上繰り返してしまった場合は、自動的に作業を停止して、ユーザーに手動操作を提案する。例：
  ```
  同じ操作を繰り返しており、問題を解決できていません。以下の手動操作をお願いします：
  [具体的なコマンドまたは手順を提示]
  ```
